Network Protocalls

HTTP
TTP, or Hypertext Transfer Protocol, is a protocol that allows for communication between web servers and web clients, such as browsers.
HTTPS
HTTPS stands for Hypertext Transfer Protocol Secure, and it's a more secure version of HTTP (Hypertext Transfer Protocol). HTTPS uses encryption to secure data transmission between a web browser and a website, ensuring that unauthorized parties can't intercept or tamper with the data.
DNS
The Domain Name System (DNS) is a hierarchical naming system that translates human-readable domain names into machine-readable IP addresses. This allows people to access information online using domain names like "nytimes.com" or "espn.com" without having to memorize IP addresses like "192.0.2.44"
